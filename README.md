两指：轻点 = 鼠标右键  
三指：  
上滑：应用全屏  
右滑：切换最新的app  
下滑：最小化窗口  
左滑：页面切换，如浏览器tab页；对应快捷键 CTRL + TAB  
四指：
上滑：调度中心  
右滑：向右切换 全屏app  
下滑：应用程序窗口（对应触控板设置里的 App Expose）  
左滑：向左切换 全屏app
